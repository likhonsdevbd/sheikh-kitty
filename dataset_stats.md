# Dataset Statistics

This document provides an overview of the datasets used to train the AI model.

## Python

*   **Source:** [Nan-Do/code-search-net-python](https://huggingface.co/datasets/Nan-Do/code-search-net-python)
*   **License:** Apache 2.0
*   **Number of Samples:** 450,740

## JavaScript

*   **Source:** [Nan-Do/code-search-net-javascript](https://huggingface.co/datasets/Nan-Do/code-search-net-javascript)
*   **License:** Apache 2.0
*   **Number of Samples:** 138,155

## TypeScript

*   **Source:** Scraped from the top 3 permissively licensed TypeScript repositories on GitHub:
    *   [vuejs/vue](https://github.com/vuejs/vue)
    *   [microsoft/vscode](https://github.com/microsoft/vscode)
    *   [yangshun/tech-interview-handbook](https://github.com/yangshun/tech-interview-handbook)
*   **License:** MIT
*   **Number of Samples:** 6,555

## Solidity

*   **Source:** [ASSERT-KTH/DISL](https://huggingface.co/datasets/ASSERT-KTH/DISL)
*   **License:** Not specified, but publicly available for research.
*   **Number of Samples:** 50,000 (from a deduplicated subset)
